#アプリ紹介ランディングページ

## 概要

架空アプリ(OHA!)の紹介ページです。

## 使用技術

- HTML

- CSS（SCSS）

- JavaScript
- swiper

## 実装内容

- レスポンシブ対応

- ハンバーガーメニュー

- スムーズスクロール

- スクロール連動アニメーション

- スライダー

- アコーディオン

- TOPへ戻るボタン

## 制作期間
3週間

## 工夫した点
- スクロールに応じてタイトルが下から表示されるようにし、ユーザーがコンテンツをスムーズに読み進められる操作感を設計しました。

- 埋め込み動画がどのデバイスで見てもアスペクト比を維持し、表示が崩れないようCSSとiframeを組み合わせてサイズを制御しました

- スライダーの標準スタイルを上書きし、サイト全体のデザインに統一感を持たせました。

- コンタクトにはフォーカスを当てボタン操作でも入力しやすい設計にしました。
